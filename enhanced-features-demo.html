API Integration Example

Enhanced Task Creation

// Create task with attachments and specifications const formData = new FormData(); formData.append('action', 'create_task'); formData.append('title', 'Design Landing Page'); formData.append('task_category', 'Design'); formData.append('specifications', JSON.stringify([{ type: 'requirement', title: 'Mobile Responsive', description: 'Must work on all devices' }])); const response = await fetch('api/tasks.php', { method: 'POST', body: formData });

Work Output Sharing

// Share work output with team const outputData = { action: 'add_work_output', task_id: 123, output_type: 'image', title: 'Final Design Mockup', description: 'High-fidelity design with interactions', visibility: 'team', external_url: 'https://figma.com/design/123' }; const result = await fetch('api/tasks.php', { method: 'POST', body: JSON.stringify(outputData) });

Integration Guide

Step-by-step instructions to integrate the enhanced task management system into your existing application.

1

Database Setup

Apply the enhanced schema updates to your existing database.

# Apply database updates mysql -u username -p database_name < database/task_enhancements.sql
2

Upload Files

Copy the enhanced API files and JavaScript components.

📁 api/tasks.php (enhanced)
📁 api/attachments.php (new)
📁 assets/js/enhanced-task-manager.js
📁 enhanced-*-dashboard.php
3

Configure & Test

Set up file permissions and test the enhanced features.

# Set permissions chmod 755 uploads/tasks/ # Test features curl -X POST api/tasks.php

Configuration Details

Required PHP Settings

; php.ini settings upload_max_filesize = 50M post_max_size = 50M max_execution_time = 300 memory_limit = 256M

Directory Structure

daily/ ├── api/ │ ├── tasks.php (enhanced) │ └── attachments.php (new) ├── assets/js/ │ └── enhanced-task-manager.js ├── database/ │ └── task_enhancements.sql └── uploads/tasks/ (create)

API Reference

Enhanced Task Creation

POST /api/tasks.php

New Parameters: task_category, requirements, deliverables, external_links, specifications

File Attachment

POST /api/attachments.php

Actions: upload, get_attachments, delete_attachment, create_share_link

Work Output Sharing

POST /api/tasks.php

Actions: add_work_output, get_work_outputs, update_progress
